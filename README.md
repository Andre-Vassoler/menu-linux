# menu-linux
INTRODUÇÃO:
Projeto acadêmico desenvolvido em Linux Shell Script, que implementa um menu interativo no terminal com diversas funcionalidades administrativas e utilitárias. O sistema permite manipulação de arquivos, gerenciamento de permissões, exibição de informações do sistema, listagem de processos, além de ferramentas simples como calculadora, conversão de temperatura e criação de anotações.

FUNÇÕES:
- Contar arquivos contendo uma string
- Alterar permissões de arquivos
- Criar diretórios
- Exibir conteúdo e detalhes de arquivos
- Mostrar informações do sistema
- Listar processos ativos
- Calculadora (soma e subtração)
- Conversão de temperatura
- Criar anotações

COMO EXECUTAR NO LINUX:
- Baixe o arquivo "menu.sh"
- No terminal, crie um diretório e acesse-o:
    mkdir menulinux  (nome de sua escolha)
    cd menulinux
- Cole o arquivo "menu.sh" dentro do diretório
- De permissão de execução:
      chmod u+x menu.sh
- Execute o script:
    ./menu.sh

TECNOLOGIAS UTILIZADAS:
- Linux
- Shell Script (Bash)
